Automated macOS development environment setup

Quick Install

Run this command in your terminal to install everything:

curl -fsSL https://smyk.la | bash

For non-interactive installation with defaults:

curl -fsSL https://smyk.la | bash -s -- --yes

With environment variables:

BOOTSTRAP_EMAIL=user@example.com \ BOOTSTRAP_NAME="Full Name" \ curl -fsSL https://smyk.la | bash -s -- --yes

What Happens

The bootstrap script will:

  1. Check prerequisites (macOS, internet, git)
  2. Install Homebrew if not present
  3. Clone the dotfiles repository
  4. Set up age encryption key (via 1Password CLI or manual input)
  5. Configure git filters for transparent encryption
  6. Install all Homebrew packages and dependencies
  7. Initialize and apply chezmoi configurations
  8. Install Vim and Tmux plugins
  9. Set up git hooks for quality checks
  10. Run verification tests

Options

Option Short Description
--yes -y Skip confirmation prompts and use defaults
--email EMAIL Set email address (overrides git config)
--name NAME Set full name (overrides git config)
--dir DIRECTORY Installation directory
--help -h Show help message
